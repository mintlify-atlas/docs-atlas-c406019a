## Report Types

<Tabs>
  <Tab title="Asset Reports">
    **Inventory and Asset Status Reports**
    
    Generate comprehensive views of your asset portfolio:
    
    - **Full Inventory**: Complete list of all assets across categories
    - **Category Breakdown**: Assets grouped by COMPUTO, INFRAESTRUCTURA, etc.
    - **Status Report**: Assets by operational status (available, in maintenance, retired)
    - **Age Analysis**: Asset age distribution and replacement forecasting
    - **Utilization Metrics**: Asset usage patterns and efficiency
    
    Example data structure:
    ```typescript
    interface AssetReport {
      totalAssets: number;
      byCategory: {
        category: string;
        count: number;
        percentage: number;
      }[];
      byStatus: {
        available: number;
        inMaintenance: number;
        retired: number;
      };
      avgAge: number;
    }
    ```
  </Tab>
  
  <Tab title="Maintenance Reports">
    **Maintenance History and Performance**
    
    Track maintenance operations and effectiveness:
    
    - **Maintenance History**: Complete log of all maintenance activities
    - **Preventive vs. Corrective**: Ratio analysis of maintenance types
    - **Cost Analysis**: Maintenance expenses by category and asset
    - **Technician Performance**: Workload and completion metrics
    - **Downtime Analysis**: Asset availability impact from maintenance
    
    Key metrics:
    - Mean Time Between Failures (MTBF)
    - Mean Time to Repair (MTTR)
    - Total maintenance hours
    - Parts and labor costs
    - Maintenance backlog
  </Tab>
  
  <Tab title="Financial Reports">
    **Budget and Cost Analysis**
    
    Monitor financial aspects of asset management:
    
    - **Budget Execution**: Actual vs. planned spending (shown on dashboard)
    - **Cost by Category**: Expense breakdown by asset type
    - **Maintenance Costs**: Service and repair expenditures
    - **ROI Analysis**: Return on investment for assets
    - **Depreciation Reports**: Asset value over time
    
    Integrates with dashboard budget metric:
    ```tsx
    <StatCard 
      title="Presupuesto ejecutado" 
      value="100$" 
      highlighted={true} 
    />
    ```
  </Tab>
  
  <Tab title="Performance Reports">
    **Operational Efficiency Metrics**
    
    Analyze system and team performance:
    
    - **Asset Availability**: Uptime percentages by category
    - **Maintenance Efficiency**: Task completion rates and times
    - **User Activity**: System usage patterns and adoption
    - **Trend Analysis**: Month-over-month comparisons
    - **Compliance Reports**: Regulatory and policy adherence
  </Tab>
</Tabs>

## StatCard Component for Reports

The reports page uses a custom StatCard implementation that differs slightly from the dashboard version:

```tsx
function StatCard({ icon, title, value, subtext, isDate = false }: any) {
  return (
    <div className="bg-white p-6 rounded-3xl shadow-sm hover:shadow-lg transition-all duration-300 border border-slate-50 group hover:-translate-y-1">
      <div className="mb-4 text-slate-700 p-3 bg-slate-50 w-fit rounded-xl border border-slate-100 group-hover:bg-gima-light group-hover:text-gima-blue transition-colors">
        {icon}
      </div>
      <h3 className="font-bold text-slate-800 mb-4 text-sm uppercase tracking-wider opacity-80">
        {title}
      </h3>
      {isDate ? (
        <div className="text-sm font-medium text-slate-600 mb-1 bg-slate-100 w-fit px-3 py-1 rounded-full">
          {value}
        </div>
      ) : (
        <>
          <div className="text-xs text-slate-400 mb-1">{subtext}</div>
          <div className="text-4xl font-microgramma text-slate-800">{value}</div>
        </>
      )}
    </div>
  );
}
```

<Note>
  The reports StatCard features hover animations (lift effect with `-translate-y-1`) and icon color transitions that enhance interactivity.
</Note>
