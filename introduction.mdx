## What is GIMA?

GIMA (Gestión Inteligente de Activos y Mantenimiento) is a comprehensive asset and maintenance management system designed to help organizations track, monitor, and maintain their physical assets efficiently. Built with cutting-edge technologies, GIMA provides real-time insights, AI-powered diagnostics, and streamlined maintenance workflows.

<Note>
  GIMA combines enterprise-grade asset management with an intuitive, modern user interface to deliver powerful capabilities without complexity.
</Note>

## Key Features

GIMA offers a complete suite of capabilities for modern asset management:

### Asset Management
- **Real-time Asset Tracking**: Monitor all your assets with live status updates and location tracking
- **Comprehensive Asset Profiles**: Store detailed information including specifications, documents, and maintenance history
- **Category Management**: Organize assets by custom categories and classifications
- **Asset Lifecycle Tracking**: Track assets from acquisition through disposal

### Maintenance Operations
- **Preventive Maintenance Scheduling**: Automate maintenance schedules based on time, usage, or condition
- **Work Order Management**: Create, assign, and track maintenance tasks with full workflow support
- **Maintenance History**: Complete audit trail of all maintenance activities
- **Parts and Inventory Management**: Track spare parts and supplies for maintenance operations

### AI-Powered Insights
- **Intelligent Diagnostics**: AI-driven analysis to predict potential failures
- **Maintenance Optimization**: Smart recommendations for optimal maintenance timing
- **Performance Analytics**: Advanced reporting on asset performance and reliability

### Security & Configuration
- **Role-Based Access Control**: Granular permissions for users and teams
- **Multi-language Support**: Interface available in multiple languages
- **Customizable Settings**: Adapt the system to your organization's needs
- **Secure Authentication**: Enterprise-grade security with modern authentication

## Who is GIMA For?

GIMA is designed for organizations that need to manage physical assets effectively:

- **Maintenance Teams**: Streamline work orders and preventive maintenance
- **Facility Managers**: Track building equipment and infrastructure
- **Operations Managers**: Monitor asset performance and availability
- **IT Administrators**: Manage technical assets and equipment lifecycle
- **Finance Teams**: Track asset costs and budget execution

## Technology Stack

GIMA is built with modern, proven technologies:

- **Frontend Framework**: Next.js 16.1.1 with React 19.2.3
- **Language**: TypeScript 5 for type-safe development
- **Styling**: Tailwind CSS 4 with custom design system
- **UI Components**: Shadcn UI components with Lucide icons
- **React Compiler**: Enabled for optimized performance
- **Architecture**: App Router with Server Components

<Warning>
  GIMA requires Node.js 20 or higher for optimal performance and compatibility.
</Warning>

## System Architecture

GIMA follows a modern, component-based architecture:

```
src/
├── app/                    # Next.js App Router pages
│   ├── auth/              # Authentication pages
│   ├── dashboard/         # Main dashboard and analytics
│   ├── configuracion/     # System configuration
│   ├── reportes/          # Reporting interface
│   └── asistencia/        # Support and help
├── components/            # React components
│   ├── layout/            # Layout components (MainLayout, DashboardHeader)
│   ├── dashboard/         # Dashboard-specific components (StatCard)
│   ├── configuracion/     # Configuration components
│   └── ui/                # Reusable UI components
├── types/                 # TypeScript type definitions
├── services/              # API and data services
├── data/                  # Static data and configurations
└── lib/                   # Utility functions and helpers
```
