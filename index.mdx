## What Makes GIMA Different

<Accordion title="Enterprise-Ready Asset Management">
  GIMA is built for enterprise-scale operations with support for unlimited assets, 
  multiple locations, and comprehensive user management with role-based access control.
</Accordion>

<Accordion title="AI-Powered Intelligence">
  Our AI assistant helps diagnose equipment issues, suggests maintenance schedules, 
  and provides intelligent insights based on your asset data and maintenance history.
</Accordion>

<Accordion title="Modern Technology Stack">
  Built with Next.js 16, React 19, and TypeScript, GIMA delivers a fast, 
  responsive experience with server-side rendering and optimized performance.
</Accordion>

<Accordion title="Customizable and Extensible">
  Configure GIMA to match your organization's needs with flexible category management, 
  custom notifications, and configurable security settings.
</Accordion>

## Getting Started

<Steps>
  <Step title="Install Dependencies">
    Set up your development environment and install all required dependencies.
    
    ```bash
    npm install
    # or
    pnpm install
    ```
  </Step>
  
  <Step title="Configure Your Environment">
    Set up your configuration files and environment variables for your deployment.
  </Step>
  
  <Step title="Start Using GIMA">
    Launch the application and begin managing your assets with the intuitive dashboard.
    
    ```bash
    npm run dev
    ```
  </Step>
</Steps>
